# Multiscale Molecular Simulations Reveal the Structural Pathway of Histatin 1 Self-Association
Data and software pertaining to the article, Multiscale Molecular Simulations Reveal the Structural Pathway of Histatin 1 Self-Association.

.ipynb files contains code for generating presented figures, this contains scripts for performing decomposition analysis methods

chain_index.sh was used to generate GROMACS index groups for each peptide chain in a simulation

auto_index.sh was used to generate GROMACS index groups for each amino acid residue in a simulation

pairdist.sh was used to generate pair distance data for a number of chains or residues
